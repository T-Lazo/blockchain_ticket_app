# Blicket — Blockchain Ticketing

Ethereum-based event ticketing dapp on Sepolia testnet.  
Contract: `0xFb98647EFe39e27A24Bd14067Af6dE182De35C96`

## Requirements

- [MetaMask](https://metamask.io) browser extension
- Sepolia ETH — get some free at [sepoliafaucet.com](https://sepoliafaucet.com)

## Run

```bash
git clone <repo-url>
cd ticketing-dapp

Go Live!
```

## Pages

| Page          | What it does                                      |
| ------------- | ------------------------------------------------- |
| Create Wallet | Generate a new Ethereum wallet                    |
| Buy Ticket    | Purchase a ticket for 0.01 SETH                   |
| Return Ticket | Return ticket and reclaim SETH                    |
| Verify Ticket | Doorman tool — check if an address holds a ticket |
